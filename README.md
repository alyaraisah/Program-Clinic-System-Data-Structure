## Latar Belakang
* Dalam era global ini, struktur data merupakan bagian yang tak terpisahkan dalam pembuatan program aplikasi yang dapat mempermudah pengolahan data. * Oleh karena itu, kami mencoba mengimplementasikan 3 materi stuktur dara, yaitu linked list, queue, dan stack dengan menuangkannya dalam sebuah Project Praktikum Struktur Data dengan judul â€œIntegrated Skin Treatment Clinic Managementâ€. Program terintegrasi yang dapat mendata _treatment_ yang dilakukan pasien klinik, mengatur perubahan data, serta mengatur proses antrian hingga pembayarannya.
